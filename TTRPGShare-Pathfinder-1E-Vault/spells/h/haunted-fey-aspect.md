## Haunted Fey Aspect

**source**:: Ultimate Combat pg. 230  
**school**:: illusion (glamer)
**level**:: arcanist 0, bard 0, medium 0, mesmerist 0, psychic 0, skald 0, sorcerer 0, wizard 0

**range**:: Personal
